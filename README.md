# Physics2D
Physics2D - Simple 2D Physics Engine for tutoring
# Build
cmake CMakeLists.txt
# Requirement
- C++ 17
- vcpkg
  - Qt
  - fmt

# Features
- Collision Detection
  - GJK/EPA
- Simple 2D Geometry Algorithm
- Rigid Body Dynamics Simulation
- Axis-Aligned Bounding Box Generator
